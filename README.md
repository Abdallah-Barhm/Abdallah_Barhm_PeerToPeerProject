# Abdallah_Barhm_PeerToPeer
مشروع نظام المحفظة الرقمية باســـــــــــتخدام تقنـــــــــــية P2P
يهدف هذا المشروع إلى إنشاء نظام محفظة رقمية متكامل يعتمد على مفهوم المعاملات الرقمية وإدارة المحافظ الإلكترونية باستخدام لغة البرمجة Python.

تم تطوير المشروع باستخدام واجهة رسومية حديثة تتيح للمستخدم إنشاء المحافظ الرقمية وإدارة الأرصدة وإرسال الأموال بين المستخدمين بسهولة، بالإضافة إلى تطبيق مفاهيم البلوك تشين والتعدين وإدارة المعاملات المالية.

يوفر النظام بيئة تعليمية وعملية لفهم كيفية عمل المحافظ الرقمية وسلاسل الكتل (Blockchain) وأنظمة الدفع الإلكترونية، مع إمكانية دعم عدة عملات رقمية داخل النظام نفسه.

كما يمثل المشروع تطبيقاً عملياً لمفاهيم البرمجة الكائنية (OOP)، وهياكل البيانات، والتعامل مع الملفات، وتصميم واجهات المستخدم الرسومية باستخدام مكتبة Tkinter.**** Systems) المستخدمة في العديد من الأنظمة الحديثة


ويستطيع المستخدم من خلال النظام:

1- إنشاء حساب مستخدم جديد.

2- تسجيل الدخول والخروج.

3- إنشاء محافظ رقمية متعددة.

4- إرسال الأموال بين المحافظ.

5- تعدين المعاملات المعلقة.

6- عرض سلسلة الكتل Blockchain.

7- استخدام العملات المتعددة.

8- إنشاء رموز QR للمحافظ.

9- حفظ سجل المعاملات المالية.

ويتم تسجيل جميع العمليات المالية داخل النظام على شكل معاملات رقمية يتم حفظها في سلسلة كتل بسيطة تحاكي أنظمة  Blockchain الحقيقية.
